# Archive

Historical notes and environment-specific helper scripts live here so the main
student path stays focused on source, configs, tests, and canonical docs.

- `archive/docs/` preserves old closeout notes, MVP planning notes, and current-paper handoff trackers.
- `archive/scripts/` preserves administrative setup scripts, live endpoint
  probes, and superseded direct helpers that are not part of the normal local
  development workflow.
